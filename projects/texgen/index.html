texgen

source: github - collaboration: wiki

texgen is a setup script for LaTeX documents.
It supports a variety of templates.

  • scientific default templates
  • custom user templates
  • custom system templates
  • normal empty document
  • arc42 standard template

the templates can be used as

  • project documentation (as doc/ folder in project)
  • standalone documentation (as new folder with given name)

supports so far only english and partially german.

contribution

this tool runs under the MIT license.
If you want to improve this script you can do so.

New options or other types of templates would be nice.
translations are also welcome
